# UI_GUIDELINES.md

## Design Principles

* Simple
* Clean
* Mobile-first
* Accessible

---

## Colors

Use a consistent design system.

Avoid hardcoded colors throughout the application.

---

## Components

Create reusable components.

Examples:

* Button
* Input
* Modal
* Card
* Select
* Table

---

## Forms

Use:

* React Hook Form
* Zod

Display validation errors clearly.

---

## Loading States

Always show loading states.

Avoid blank screens.

---

## Responsive Design

Support:

* Mobile
* Tablet
* Desktop

Mobile experience is mandatory.
